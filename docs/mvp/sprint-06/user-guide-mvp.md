# User Guide MVP

## Login
- Buka aplikasi dan login sesuai role: admin/guru/siswa.

## Admin
- Kelola user dan kelas.
- Pantau dashboard ringkasan tenant.

## Guru
- Buat soal di Bank Soal.
- Buat ujian dari menu Exams.
- Attach soal, run precheck, publish.
- Lihat laporan class results dan export CSV.

## Siswa
- Buka My Exams.
- Start ujian dalam jadwal aktif.
- Jawab soal, submit, lihat hasil.

## Notifications
- Cek menu notifications di topbar.
- Gunakan Read/Read All untuk update status.

## PWA
- Jika browser mendukung, klik tombol Install App di topbar.

## Troubleshooting
- Jika endpoint baru 404, rebuild API container.
- Jika akses ditolak, pastikan token login masih valid.
