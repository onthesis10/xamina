# Pilot Test Plan (Sprint 6)

## Objective
Memvalidasi flow end-to-end di 1 sekolah pilot tanpa bug kritis.

## Participants
- 1 admin sekolah
- 2 guru
- 30-50 siswa

## Scenario Coverage
1. Login admin/guru/siswa.
2. Guru buat soal + ujian + publish.
3. Siswa kerjakan sesi hingga selesai.
4. Dashboard, reports CSV, notifications.
5. PWA install prompt dan akses ulang app.

## Acceptance Gate
- Tidak ada P1/P2 open.
- Minimal 95% siswa selesai ujian tanpa blokir.
- Hasil nilai dan laporan konsisten.
- Feedback terkumpul ke log template.

## Evidence
- Rekaman issue/finding per skenario.
- Screenshot endpoint/report penting.
- Ringkasan keputusan go/no-go.
