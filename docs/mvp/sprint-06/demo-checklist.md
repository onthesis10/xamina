# MVP Demo Checklist

## Pre-demo
- [ ] API healthy (`/health`)
- [ ] Metrics healthy (`/metrics`)
- [ ] Dashboard data muncul untuk admin/guru/siswa
- [ ] Reports CSV export jalan
- [ ] Notifications read/read-all jalan
- [ ] Student session start->finish->result jalan
- [ ] PWA install prompt terpicu di browser supported

## Demo Script
1. Admin login dan cek dashboard.
2. Guru create/publish exam.
3. Siswa start exam, submit, lihat result.
4. Guru/admin export report CSV.
5. Tunjukkan notifikasi dan PWA install.

## Post-demo
- [ ] Catat feedback
- [ ] Klasifikasi bug P1/P2/P3
- [ ] Tentukan action next sprint
